# Audio Folder

Drop royalty-free MP3 tracks here:
- `calm.mp3` — slow, ambient/lo-fi (study-focus vibe)
- `fast.mp3` — chiptune/electronic (boss-battle vibe)

## Free Sources
- [Pixabay Music](https://pixabay.com/music/) — free, no attribution required
- [Free Music Archive](https://freemusicarchive.org/) — varies by track
- [OpenGameArt](https://opengameart.org/art-search-advanced?field_art_type_tid%5B%5D=12) — game-specific

## Fallback
If these files are missing, the game falls back to procedurally-generated 8-bit synth tunes
(triangle-wave chord pad for Calm; square-wave arpeggio + sawtooth bass for Fast).
